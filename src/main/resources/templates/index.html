About Us

The Employee Data Management System (EDMS) is a streamlined solution designed to handle the essential task of managing employee records within an organization. The system centralizes employee data, such as personal information, job roles, contact details, department, and work history. This system will provide an intuitive user interface for HR staff to add, update, and retrieve employee information.
